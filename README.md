# cloud-itonami-iso3166-prt

Open ISO 3166 Blueprint for **PRT**: Portuguese Republic.

Independent public-sector market-entry & procurement-compliance service
for an already-incorporated operator entering public contracts in Portugal.

## Official surface (web-verified)

- Procurement: BASE.gov.pt / Vortal public e-procurement under CCP (Código dos Contratos Públicos)
- Business/tax: NIPC/NIF via Autoridade Tributária e Aduaneira + commercial registry (RNPC)

## What this is NOT

- **Not the government of Portugal.**
